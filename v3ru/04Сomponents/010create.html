Создание компонентов

Для создания компонента у объекта приложения Vue используется метод component(имя_компонента, конфигурация_компонента),
где первый параметр представляет имя компонента,
а второй параметр представляет конфигурацию компонента, которая собственно и определяет его данные и поведение.

Здесь определен компонент для элемента hello. И не важно, что такого элемента в html не существует, мы его сами создаем.
Именно на место элемента hello будет впоследствии рендерится компонент Vue.
Более того мы не можем использовать для компонента встроенные элементы html типа div или h2.
