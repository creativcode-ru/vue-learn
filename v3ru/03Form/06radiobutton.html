Переключатели (radiobutton)

При установке привязки, также как и для элементов input type="checkbox", директива v-model использует его атрибут checked и событие change для отслеживания изменения значения, а начальное значение атрибута checked.
Переключатели привязываются к одному свойству. При выборе переключателя связанное свойство получает выбранное значение:




Выбрано: {{ user }}
