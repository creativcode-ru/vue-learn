Иногда возникает необходимость определить в компоненте несколько слотов, которые могли бы иметь разное содержимое. В этом случае каждому слоту можно назначить определенное имя и через это имя передавать в него некоторый контент из родительского компонента. Для этого в элементе slot используется атрибут name.

В родительском компоненте для слотов передается содержимое, которое оборачивается в элемент template. У этого элемента атрибут v-slot принимает имя слота, на место которого необходимо вставить содержимое.
